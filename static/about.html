MindJotter is a web-based journaling application that enables users to write personal journals anytime, anywhere.


The only requirement to use MindJotter is an internet-enabled device. MindJotter aims to empower users to develop more positive state of mind through journaling.


MindJotter leverages best-in-class large language model (llama3 7B) to generate meaningful insights for users, based on their journal entries.
